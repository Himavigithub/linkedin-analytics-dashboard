# 🔗 LinkedIn Analytics Dashboard

## 📊 Project Overview

This project analyzes LinkedIn profile performance using Power BI. It provides insights into engagement, audience distribution, and growth trends.

---

## 🚀 Features

* KPI metrics (Followers, Impressions, Profile Views)
* Engagement Analysis (Donut Chart)
* Audience Insights (Industry & Location)
* Growth Trend (Line Chart)
* Engagement Funnel Analysis

---

## 🛠️ Tools Used

* Power BI
* Excel
* Data Visualization Techniques

---

## 📈 Key Insights

* Identified engagement drop from impressions to followers
* Analyzed audience distribution by industry and location
* Evaluated content performance using engagement metrics
